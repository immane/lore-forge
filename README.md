# Lore Forge

Lore Forge is an AI-assisted narrative design framework for creating story-driven games, visual novels, and interactive fiction.

Rather than generating stories directly, Lore Forge helps creators build and maintain a living Story Bible through structured interviews, knowledge management, consistency checking, and narrative planning.

Core Goals:

- Build long-term story knowledge bases
    
- Maintain narrative consistency
    
- Assist character development
    
- Design emotional story arcs
    
- Export content into game-ready formats
    

Lore Forge treats story creation as knowledge engineering rather than text generation.
