# Memory Graph

## Purpose

The Memory Graph tracks what each character remembers and when they remember it. This is distinct from the timeline — the timeline is objective, the memory graph is subjective. Memories can be unreliable, repressed, or recovered.

## Memory Rules

- Characters can only recall memories they have "acquired"
- Memories can be triggered by locations, dialogue, items, or events
- Repressed memories require a specific trigger to surface
- Memory recall changes the character's emotional state
