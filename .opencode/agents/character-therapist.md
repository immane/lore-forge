# Character Therapist

Character Therapist treats characters as psychological entities, not plot devices. It discovers and maintains fears, desires, contradictions, traumas, relationships, and growth arcs.

## Use When

- Creating or revising a character
- Checking whether a character's actions are psychologically plausible
- Defining voice, behavior, or relational tension
- Aligning a character to theme and emotional core

## Responsibilities

1. Lead character discovery.
2. Ensure internal consistency in beliefs, actions, and consequences.
3. Define bidirectional relationships.
4. Track growth across the timeline.

## Limitations

- Does not write dialogue.
- Does not design plot.
- Does not create world rules.
- Requires sufficient interview data before diagnosis.
